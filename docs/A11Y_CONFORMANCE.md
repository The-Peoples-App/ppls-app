# Accessibility Conformance Statement: The Peoples App
## Commitment Statement
Hunger Action Los Angeles (HALA) is dedicated to ensuring digital equity and dignified access for all Angelenos, including Seniors and people with disabilities. We recognize that accessibility is both a legal obligation and a core organizational value in the fight against food insecurity.

## Conformance Status
The Peoples App is architected to be **Partially Conformant** with WCAG 2.1 Level AA standards, with specific high-priority features meeting **Level AAA** criteria for enhanced contrast and typography.

- **Primary Standard**: WCAG 2.1 Level AA / AAA
- **Secondary Standard**: Section 508 of the Rehabilitation Act

## Technological Foundation
To achieve these rigorous standards, we utilize the **Google Material Design 3 (M3)** ecosystem, which provides a pre-engineered foundation for inclusive design.

- **Semantic Structure**: All components utilize standard platform controls to ensure seamless compatibility with assistive technologies like **VoiceOver** (iOS) and **TalkBack** (Android).

- **Visual Equity**: We leverage the **M3 Tonal Palette** to programmatically ensure **Level AAA contrast ratios (7:1)** for all core advocacy content, specifically benefiting our Blind and Low-Vision (BLV) user base.

## Evaluation & Validation
Our conformance is verified through a combination of automated testing and manual human evaluation, ensuring that the "to-the-point" information in the People's Guide is accurately relayed to all users.

- **Internal Audits**: Conducted using the [W3C Accessibility Statement Generator](https://www.w3.org/WAI/planning/statements/) and official WCAG Quick Reference tools.
